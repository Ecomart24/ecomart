Email Configuration Setup

Follow these steps to enable email sending for your address card system

1

Create EmailJS Account

Go to EmailJS.com and sign up for a free account.

2

Create Email Service

In your EmailJS dashboard, add an email service (Gmail, Outlook, etc.).

  • Click "Email Services" → "Add New Service"
  • Choose your email provider (Gmail recommended)
  • Connect your email account
3

Create Email Templates

Create two email templates in EmailJS:

Template 1: OTP Email
Subject: Your OTP Verification Code Hello {{user_name}}, Your OTP code is: {{otp}} This code will expire in 10 minutes. Thank you!
Template 2: Address Card Email
Subject: Your Custom Address Card Hello {{fullName}}, Here is your custom address card: {{card_html}} Thank you for using our service!
4

Get Your Credentials

Find your credentials in the EmailJS dashboard:

  • Public Key: Account → API Keys → Public Key
  • Service ID: Email Services → Your Service → Service ID
  • Template IDs: Email Templates → Your Templates → Template ID
5

Update the Code

Replace the placeholder values in address-card.html:

// Find this line in address-card.html: emailjs.init("YOUR_PUBLIC_KEY"); // Replace with your actual Public Key emailjs.init("YOUR_ACTUAL_PUBLIC_KEY_HERE");
// Find the sendOTPEmail function and replace with: await emailjs.send('YOUR_SERVICE_ID', 'YOUR_OTP_TEMPLATE_ID', { to_email: email, user_name: formData.fullName, otp: otp });
// Find the sendAddressCardEmail function and replace with: await emailjs.send('YOUR_SERVICE_ID', 'YOUR_CARD_TEMPLATE_ID', { to_email: data.email, fullName: data.fullName, card_html: cardContent });
Quick Setup Alternative

If you want to test immediately without EmailJS setup, the current code will show the OTP in an alert box and log the card details to the console. This is perfect for testing the functionality!
